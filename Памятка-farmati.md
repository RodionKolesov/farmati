# Памятка по сайту farmati.ru — что сделать дальше

Сайт уже запущен и работает: **https://farmati.ru** (HTTPS, админка готова).
Ниже — что осталось сделать и добавить, по порядку важности.

## 1. Сразу (обязательно)
- Зарегистрировать администратора: на сайте «Регистрация» с email **admin@farmati.ru** и надёжным паролем → появится доступ в админку /admin.
- Сменить root-пароль сервера (он показывался в переписке): по SSH команда passwd.
- Перезагрузить сервер, когда удобно (висит сообщение про обновление ядра) — сайт поднимется сам.

## 2. Наполнить контентом (всё через админку /admin)
- Товары: реальные фото, цены и остатки (склад). Если остаток 0 — товар скрывается автоматически.
- Курсы и уроки: вставить ссылки на видео (YouTube/VK), отметить первые 3 урока бесплатными.
- Чек-листы: загрузить PDF в разделе «Чек-листы».
- Отзывы: добавить фото «до/после» в разделе «Отзывы».
- Заменить демо-данные (для примера у первого товара стоит остаток 2 шт.).

## 3. Чтобы принимать деньги
- Подключить эквайринг ЮKassa (договор + ключи). Без него оплата работает в тестовом режиме (заказ оформляется, но деньги не списываются).
- Определиться с процентом кэшбэка: 5% или 7% (меняется в настройке .env).

## 4. Уведомления и удобство
- Telegram-уведомления о новых заказах и заявках — бесплатно, рекомендуется.
- По желанию: вынести тексты главной страницы в админку, чтобы менять без программиста.

## 5. Юридическое (152-ФЗ) — черновики уже готовы
- Подать уведомление в Роскомнадзор.
- Опубликовать на сайте Политику конфиденциальности.
- Добавить согласие на обработку персональных данных (галочка на формах регистрации/заказа).
- Согласие на рекламную рассылку — если будете делать рассылки.
- Черновики на рабочем столе: «РКН-уведомление-черновик.docx», «Инфо-о-проекте-для-РКН.docx».

## 6. На будущее (когда вырастете)
- Регулярно делать бэкап базы: файл /root/farmati/web/prod.db (скачивать через WinSCP).
- Перейти на платный сервер (2 ГБ, ~590 ₽/мес) при росте трафика или нехватке места.
- SMS-уведомления клиентам — при необходимости (платный сервис).

На сервере по SSH: cd /root/farmati && git pull && cd web && npm ci && npx prisma db push && npm run build && pm2 restart farmati
